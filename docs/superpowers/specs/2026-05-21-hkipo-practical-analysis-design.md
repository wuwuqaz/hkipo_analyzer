# 港股IPO实战分析模块改造设计文档

> **日期**: 2026-05-21
> **状态**: 已批准
> **背景**: 业界研究表明，港股IPO分析在实战层面（中签率预测、甲乙组策略、定价价差、绿鞋机制）是关键决策因子，本项目在这些维度存在短板。

---

## 一、改造目标

提升项目的**打新实战指导能力**，从"基本面分析优秀"升级为"实战决策利器"。

## 二、改造范围（5个模块）

### 🔴 高优先级

| # | 模块 | 文件 | 核心功能 |
|---|------|------|---------|
| 1 | 定价价差增强 | `scoring.py` + `analyzers/_pricing_gap.py` | 将定价位置提升为独立评分维度 |
| 2 | 中签率预测 | `allotment_predictor.py` | 基于超购倍数预测一手中签率 |
| 3 | 甲乙组策略 | `a_b_group_strategy.py` | 甲头/甲尾/乙头最优策略分析 |

### 🟡 中优先级

| # | 模块 | 文件 | 核心功能 |
|---|------|------|---------|
| 4 | 绿鞋机制分析 | `greenshoe_analyzer.py` | 绿鞋检测与稳价影响评分 |
| 5 | 行业赛道热度 | `sector_heat.py` | 同板块新股近期表现分析 |

### 3.2 集成点

```

                                    ↓

                                                   ↓
over_sub_ratio → allotment_predictor.predict() → 预测中签率 → 前端展示
```

## 四、评分体系变更

### 4.1 定价价差权重提升

**当前**：作为 peer_adj 小修正项（±1-2分）

**改造后**：独立评分维度
- 上限定价：+3分（直接加到最终评分）
- 中上限：+1分
- 中间价：0分
- 中下限：-2分
- 下限：-5分

### 4.2 绿鞋机制加分

在 `_calc_trade_score` 中：
- 有绿鞋：+2分（到 trade_max）
- 无绿鞋：0分

## 五、中签率预测算法

### 5.1 超购倍数映射表

| 超购倍数 | 一手中签率 | 依据 |
|---------|-----------|------|
| < 5倍 | 80-100% | 雪球/TradeSmart 数据 |
| 5-15倍 | 50-80% | 2026年港股统计 |
| 15-50倍 | 20-50% | 正常热度 |
| 50-100倍 | 6-20% | 热门区间 |
| 100-500倍 | 2-10% | 极热区间 |
| 500-1000倍 | 0.5-3% | 疯狂区间 |
| > 1000倍 | 0.03-1% | 蜜雪/雪王等极端案例 |

### 5.2 稳中一手所需资金

基于超购倍数和公开发售手数计算：
- 超购 < 15倍：约 50-150万 可稳中
- 超购 15-50倍：约 150-300万
- 超购 50-100倍：约 300-500万
- 超购 > 100倍：500万+（需乙组）
